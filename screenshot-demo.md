# Lark-Style Theme Screenshot Demo

Use this note to capture screenshots for the Obsidian theme listing.
Open it in Obsidian with Lark-Style theme enabled, then capture screenshots.

## Recommended screenshots to capture

1. **`lark-style-light.png`** — Light mode, full workspace (file tree + editor with mixed content)
2. **`lark-style-dark.png`** — Dark mode, same view

## How to capture

- Obsidian → View → Toggle light/dark mode
- Cmd + Shift + 4 (macOS region screenshot), or use a window-screenshot tool
- Resize Obsidian window to ~1440×900 before capturing (standard ratio)
- Screenshot the whole window (include file tree + main editor)
- Save as PNG to this folder

## Test content to display during screenshot

Open the file `demo-vault.md` (in this folder) for a richly-styled note that showcases all theme features.

## Obsidian Theme Listing requirements

From https://docs.obsidian.md/Themes/Obsidian+Theme+submission+guidelines:
- Minimum one screenshot
- PNG format, ideally 1440×900 or larger
- Show the theme in normal use (not just a blank screen)
- Both light and dark modes recommended
